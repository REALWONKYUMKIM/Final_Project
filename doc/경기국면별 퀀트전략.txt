제목: 경기국면별 퀀트전략(동적자산배분 포트폴리오 추천

- 개요: 시장의 다양한 경기국면에서 가장 잘먹히는, 가장 잘 안먹히는 퀀트전략(동적자산배분) 전략을 찾고 현재와 가까운 미래에는 어떤 전략이 잘 통할지 예측을 해본다. 

- 사용할 데이터셋: 올웨더때와 비슷하게 미국 주식/채권/원자재/금과 같은 etf 사용 

- 사용될 퀀트전략(동적자산배분): 
	* 전통듀얼모멘텀전략 (개리 안토나치 -- 논문있음) 
	* LAA(영구 포트폴리오 업데이트 버전)/RAA(올웨더 업그레이드)/DAA (Wouter Keller 논문있음)


- 비교방법 런다운: 
 1.French/Fama 경기국면(4국면)으로 나누고 거기에서 현재 우리가 속한 경기국면이 어디인지 확인 
 2. 위에 소개된 4개 전략이 각각의 경기 국면에서 어떤 성과를 냈는지에 대해서 확인 
 3. 각각의 전략의 CAGR, MDD, Sharpe, Sigma 분석 후 어떤 모델이 가장 수익률이 높고, 적고/ MDD가 가장 높고, 낮고 //Sharpe 지수가 높고 낮고 // Sigma가 높고, 낮고 확인 

 4.현재 시장상황 ('23년 2/4분기)에 위에 4가지 전략중 어떤 전략이 잘 먹힐지 확인 


- 결론: 현재 ('23년 2/4분기에서는 어떤 퀀트전략이 잘먹히고 또 시나리오별로 미래에 다양한 경기국면에서 어떤 전략을 써야할지 추천.

* 절대 모멘텀 전략: 자산의 현재의 가격과 과거의 가격을 비교하여 상승하였을 경우 매수, 하락하였을때 매도 
* 상대 모멘텀 전략: 여러 자산군간의 모멘텀을 비교하여 더 많이 오른 자산군을 매수하는 전략
* 듀얼 모멘텀: 절대모멘텀 전략+상대모멘텀 전략 == 최근 12개월 중 가장 모멘텀이 좋은(주가변동성이 가장 큰)자산을 매수하되 하락장에서는 전량매도 후 현금보유
